Getting Started

Welcome

MIA is a premium one-page HTML5/CSS3 website template, based on Twitter Bootstrap 3.2.0 framework.

MIA is designed with agencies and small teams in mind and can help them to launch a simple, modern and professional website in no time!

Features

  • Clean, Responsive & Minimal Design
  • Simple slide menu with click to scroll functionality
  • Beautiful animations and effects during scrolling down
  • Google Fonts
  • Cross Browser Compatible
  • Owl Carousel integration
  • Custom scrollbar
  • Pricing Tables
  • Easy separation and re-arrangement of each content section
  • Full documentation
  • Based on Bootstrap 3.2.0
  • jQuery Powered

Editing Texts

Each section of the html is given a unique ID in order to have the scrolling to feature active.

When you edit, add or delete a section, you must edit two different areas of the html file

1. Sidebar Slide Menu

<nav class="pull">
	<ul class="top-nav">
		...
		<li><a href="#intro">Introduction <span class="indicator"><i class="fa fa-angle-right"></i></span></a></li>
		... 
	</ul>
</nav>

2. Main documentation content

<section class="intro text-center section-padding" id="intro"> 
	<div class="container"> 
		<div class="row"> 
			<div class="col-md-8 col-md-offset-2 wp1"> 
				<h1 class="arrow">A CLEAN WEBSITE TEMPLATE, THAT LOOKS DAMN GOOD!</h1> 
				<p><strong>MIA</strong> is a premium one-page HTML5/CSS3 website template, based on Twitter Bootstrap 3.2.0 framework. MIA is designed with agencies and small teams in mind and can help them to launch a simple, modern and professional website in no time! It is really easy to use and configure and includes <a href="#">full documentation</a>.</p> 
			</div> 
		</div> 
	</div> 
</section>

Each section is clearly separated from the others with comments.

Change Colors

UniDocs has a separate colors.css file where you can find and change the colors of the templete easily.

h1 {
  color: #353535;
}

a {
  color: #353535;
}

a:hover, a:focus {
  color: #1abc9c;
}

p {
  color: #8c9398;
}

h2 {
  color: #353535;
}

h2::after {
  border-bottom: 1px dotted #8c9398;
}

the only thing you have to do, besides editing the colors.css file, is to include it in the index.html head right below styles.min.css

...
<link href="css/styles.min.css" rel="stylesheet">
<link href="css/colors.css" rel="stylesheet">
...
